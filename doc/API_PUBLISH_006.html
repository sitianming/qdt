登录
path /ms/api/v1/user/login
method POST
contentType application/json 
req:
{
    "userName": "sitianming",
    "passwd": "123456",                 // md5加密后传输
}

resp:
{
    "errcode": 0,
    "msg": "succ",
    "data": {
        "token": "xxxxxx",
        "userRole": 1,                  // 1: 普通用户, 9: 管理员
        "isDefault": true,              // 是否是初始密码
        "nickName": "我",
    },
}

{
    "errcode": 40001,
    "msg": "账号或密码错误",
}

{
    "errcode": 40002,
    "msg": "失败次数过多, 请稍后再试",
}

注:
    token包含信息 UserId, UserRole, Timestamp, 8小时过期
    token传递方式: 头信息内传递 header["X-Token"]=token
重置密码
path /ms/api/v1/user/resetPasswd
method PUT
contentType application/json 
header: {
    "X-Token": token,
}

req:
    {
        "passwd": "1234567",            // 前端md5加密
    }

resp:
    {
        "errcode": 0,
        "msg": "succ"
    }

    {
        "errcode": 40010,
        "msg": "不能与初始密码一致",
    }

    {
        "errcode": 40011,
        "msg": "只有初始密码才能重置",
    }


只有passwd为初始密码, 才能被重置密码, 成功后直接登录进入首页
前端检查密码强弱, 显示给用户看下, 密码最少6位, 最多20位 字符长度

修改任务运行时参数
path /ms/api/v1/task/modifyRunningParams
method PUT
contentType application/json 
header: {
    "X-Token": token,
}

req:
    {
        "taskId": 1,
        "params": {
            "pricing": 100000.0,        // 定价(>0)         // 普通用户可修改
                                                           // 下面都是管理员可修改
            "spread": 1.2,              // 范围限制(0~1)     // 单边spread
            "priceMultiple": 1.1,       // 范围限制(>0)      // 价格倍数
            "initOffset": -1000,                            // 初始持仓偏移
            "24hQty": 100000,           // 范围限制(>0)      // 画线24小时量
            "poStep": 120,              // 范围限制(>0)      // 画线间隔(s)
        },                              // 改哪些就传哪些, 不需要修改的就不传递
    }

resp:
    {
        "errcode": 0,
        "msg": "succ"
    }

    {
        "errcode": 40200,
        "msg": "参数格式存在不正确项",
    }
